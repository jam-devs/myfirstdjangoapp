Setting up the development environment for Django was both exciting and challenging. Initially, I struggled with activating the virtual environment because I was using the wrong command for my operating system. After reviewing documentation and watching a tutorial, I corrected the command and successfully activated it.

Another hurdle was installing Django. I encountered compatibility issues with my Python version, which I resolved by upgrading Python and reinstalling Django using pip. Running the server (`python manage.py runserver`) failed at first due to missing migrations, but I fixed it by running `python manage.py migrate`.

Lastly, configuring Git was tricky. I had trouble linking my local repo to GitHub due to authentication errors. I solved this by listening the instruction of sir Paragas
the process of how to setup the github and to commit my codes to github

Overall, the setup process taught me patience and the importance of reading error messages carefully. Each challenge helped me understand the development workflow better.
